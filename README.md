# 【講義】アルゴリズムとデータ構造

このリポジトリには、講義「アルゴリズムとデータ構造」に関するプログラムをまとめています。


## プログラム概要

### Python プログラム

`./example_python` 以下では、それぞれ次のような項目の Python プログラムが含まれています。
各 Python プログラムの詳細は、ソースコード先頭の DocString を参照してください。

1. アルゴリズムの基本構造 (`./example_python/01_basic_alg_flow`)
    - 順次構造
    - 選択構造
    - 反復構造

2. データ構造基礎 (`./example_python/02_data_struct_A`)
    - 配列
    - リスト（連結リスト）
    - スタック
    - キュー
    - ハッシュテーブル

3. データ構造応用 (`./example_python/03_data_struct_B`)
    - グラフ（無向・有向・重みなし・重みつき）
    - 木構造
    - 二分木
    - 二分探索木
    - 木構造版（最大・最小）ヒープ
    - 配列版（最大・最小）ヒープ

4. 探索アルゴリズム (`./example_python/04_search`)
    - 線形探索
    - 二分探索
    - ハッシュ探索
    - 幅優先探索
    - 深さ優先探索

5. ソートアルゴリズム (`./example_python/05_sort`)
    - 選択ソート
    - 挿入ソート
    - バブルソート
    - マージソート
    - クイックソート
    - 木構造版ヒープソート
    - 配列版ヒープソート
    - 計数ソート（バケットソート・ビンソート）
    - 基数ソート

6. 文字列パターン照合 (`./example_python/06_str_matching`)
    - 素朴な文字列パターン照合
    - KMP (Knuth–Morris–Pratt) 法
    - BM (Boyer-Moore) 法

7. 再帰処理 (`./example_python/07_recursive_alg`)
    - 再帰処理による階乗計算
    - ハノイの塔

8. アルゴリズムの応用例 (`./example_python/08_application`)
    - エラトステネスのふるい
    - ユークリッドの互助法
    - k-meansクラスタリング
